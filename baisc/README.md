This repository contains resources and code examples for beginners to learn the basics of Python programming and important data structures. It covers the fundamentals of Python syntax, data types, and more advanced topics like various data structures used in programming.

## üìÇ Contents

### 1. Python Basics:
- **Variables and Data Types**: Understanding different types such as `int`, `float`, `str`, `bool`, etc.
- **Conditional Statements**: `if`, `elif`, and `else` statements to control the flow of your program.
- **Loops**:
  - `for` loops for iterating over collections.
  - `while` loops for executing blocks of code repeatedly.
- **Functions**: Defining and using functions to modularize code.
- **Input/Output**: Handling user input and displaying output.
- **Error Handling**: Using `try`, `except` blocks to handle exceptions gracefully.

### 2. Data Structures:
- **Lists**: Creating and manipulating dynamic arrays.
- **Tuples**: Immutable sequences of items.
- **Dictionaries**: Key-value pairs for storing data efficiently.
- **Sets**: Unordered collections of unique elements.
- **Stacks**: LIFO (Last In First Out) structure using lists.
- **Queues**: FIFO (First In First Out) structure using the `collections` module.
- **Linked Lists**:
  - Singly Linked Lists
  - Doubly Linked Lists
- **Trees**:
  - Binary Trees
  - Binary Search Trees
- **Graphs**: Implementing adjacency list and adjacency matrix.
